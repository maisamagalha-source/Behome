Prompt para Figma Make — Landing Page Balada Erótica

Crie uma landing page comercial premium, mobile first e responsiva para desktop, para apresentação do projeto Balada Erótica, uma plataforma digital criada para estabelecimentos do mercado liberal que desejam mais controle, mais clientes, mais recorrência, monetização e menos dependência de redes sociais tradicionais.

A página deve parecer uma experiência de apresentação comercial viva, dinâmica e envolvente, não um site comum. Ela precisa contar uma história com começo, meio e fim, guiando o visitante por uma narrativa visual clara: primeiro mostrar o problema real do mercado, depois apresentar o Balada Erótica como solução, depois mostrar os módulos de valor, depois monetização, automação, crescimento e por fim uma chamada forte para adesão.

A identidade visual deve usar como base preto, branco e vermelho, com composição sofisticada. Pode usar variações inteligentes como vermelho vinho, vermelho neon suave, grafite, off-white, cinza chumbo, preto profundo, blur vermelho, vidro fosco, sombras densas, bordas luminosas e detalhes premium. A estética deve ser adulta, elegante, tecnológica, sensual de forma sofisticada, sem vulgaridade. Deve transmitir controle, desejo, exclusividade, liberdade e poder de negócio.

A landing page deve ter alto nível visual, parecida com sites modernos de IA, SaaS premium, fintechs e plataformas interativas. Usar movimentos, microinterações, cards animados, profundidade, parallax, elementos flutuantes, gradientes escuros, linhas finas, glow vermelho, módulos em camadas, mockups de aplicativo e seções com transição cinematográfica.
